# ABP VIN Decoder

A utility for decoding and parsing Vehicle Identification Numbers (VINs) with validation and detailed component extraction.

## Features

- VIN validation and parsing
- Decode manufacturer, model year, and serial information
- Extract check digit validation
- Support for standard 17-character VINs

## Installation

```bash
npm install abp-vin-decoder
```

## Usage

```bash
npm run dev
```
